# 🚀 DataSphere AI

A modern and responsive Database Management Portal built using PHP, MySQL, Bootstrap 5, JavaScript, and Chart.js. DataSphere AI allows users to create dynamic datasets, manage records, import/export data, generate reports, and administer users through a clean dashboard.

## 🚀 Installation

1. Clone the repository
2. Import the `datasphere.sql` database into MySQL
3. Configure database credentials in `config/database.php`
4. Install Composer dependencies

```bash
composer install
```

5. Start Apache & MySQL using XAMPP
6. Open:

```
http://localhost/Database-Portal
```
