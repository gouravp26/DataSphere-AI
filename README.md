# 🚀 DataSphere AI

A modern Database Management Portal built using PHP, MySQL, JavaScript, Bootstrap 5, and Chart.js.

## ✨ Features

- 🔐 User Authentication
- 📊 Interactive Dashboard
- 👨‍🎓 Student Management
- 👨‍🏫 Faculty Management
- 🏢 Department Management
- 📈 Analytics Dashboard
- 🌙 Dark Mode
- 📱 Responsive Design
- 📤 Export Reports
- 📥 Import Data

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- PHP
- MySQL
- Chart.js
- Git
- XAMPP
